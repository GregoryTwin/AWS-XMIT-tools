Function: Функция: 
  Anonomyze XMIT (TRANSMIT, IDTF) image - updates origin node, origin userid, origin data set name. Анонимизация (TRANSMIT, IDTF) файла - замена имени узла, имени пользователя, имени набора данных.
 
Syntax:Формат: 
 
  REXMIT [options] input output
 
Options:Режимы: 
 -node node
-userid userid
-dsname dsname
 
PrerequestedТребования: 
 
  • Either Open object REXX (OOREXX) or Regina REXXOpen Object REXX (OOREXX) либо Regina REXX
  • File extension .rex must be properly associatedРасширение имени файлов .rex должно быть правильно ассоциировано
 
InstallationУстановка 
  Installation procedure not required. Какая-либо специальная установка не требуется.
 
Usage:Применение 
  Procedure UNXMIT has two mandatory parameters: input - an original XMIT file and output - processed XMIT file. At least one option should be specified (otherwise there is nothing to do) Процедура UNXMIT имеет два обязательных параметра input - имя оригинального XMIT файла и output - имя обработанного файла. Должен быть уазан хотя бы один режим, так как иначе нечего преобразовывать
