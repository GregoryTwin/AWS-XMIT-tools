Function: Функция: 
  List and load member(s) from IEHMOVE image. Список разделов и их загрузка из файла, выгруженного IEHMOVE.
 
Syntax:Формат: 
 
  UNMOVE [options] iehmove [{member|*} [rename]]
 
Options:Режимы: 
 -binary | -text
-ebcdic codepage
-ascii codepage
 
PrerequestedТребования: 
 
  • Either Open object REXX (OOREXX) or Regina REXXOpen Object REXX (OOREXX) либо Regina REXX
  • File extension .rex must be properly associatedРасширение имени файлов .rex должно быть правильно ассоциировано
 
InstallationУстановка 
  Installation procedure not required. Какая-либо специальная установка не требуется.
 
Usage:Применение 
 
  1. Procedure UNMOVE has single mandatory parameter iehmove - name of IEHMOVE image. An optional 2nd parameter could specify member to be processed or * to process all members, 3rd parameter may specify member rename pattern.
    When 2nd and 3rd parameters are missed procedure lists all member names from IEHMOVE image file:     Процедура UNMOVE имеет один обязательный параметр iehmove имя файла с образом набора данных, выгруженного IEHMOVE. Второй параметр может указывать имя обрабатываемого раздела или *, что указывает обработку всех разделов, третий параметр указывает шаблон переименования разделов.
    Если второй и третий параметры отсутствует, процедура отображает список разделов выгруженного библиотечного набора данных:
    F:\>unmove.rex JESSY.INSTALL
    021I Data set JESSY.INSTALL from SVSKKK
    022I Data set attributes: DSORG=PO,RECFM=FB,LRECL=80,BLKSIZE=800
    020I Member COPYPROC TTR 000000
    020I Member INSTALL TTR 000302
    020I Member MVT5050 TTR 000803
    020I Member MVT5061 TTR 000406
    020I Member MVT5066 TTR 00030D
    020I Member PRTINSTR TTR 000502
    020I Member SERVLOC TTR 000706
    020I Member SVS5050 TTR 00020F
    020I Member SVS5061 TTR 000704
    020I Member SVS5066 TTR 000608

  2. File name of the extracted member(s) or data set build using rename pattern, where %M% substituted with original member name and %D% substituted with original data set name. If rename pattern is not specified, default pattern %M% is used for partitioned data set and %D% for other data sets. So, extracted members and a data sets would keept their original names. Имя файла с извлекаемым разделом (разделами) формируется на основании шаблона переименования, при этом %M% заменяется именем оригинального раздела а %D% - именем оригинального набора данных. Если шаблон переименования не был указан, по умолчанию для библиотечных наборов данных используется шаблон %M%, и именем файла будет оригинальное имя раздела, а для наборов других организации %D%, т.е. имя оригинального набора данных.
  3. Data translation. If option -text specified (this is the default option), data would be translated from EBCDIC to ASCII. The options -ascii and -ebcdic may be used to specify a particular code pages for translation. If both -ascii and -ebcdic are missed, an internal translation table is used. This internal table specified 1025 to 1251 translation.
    If option -binary specified, original data remains unchanged.     Перекодировка данных. Если указан режим -text (этот режим активен по уолчанию), option), данные транслируются из кода EBCDIC в код ASCII. Режимы -ascii и -ebcdic могут использоваться чтобы определить, какие именно кодовые таблицы используются при перекодировке. Если режимы -ascii и -ebcdic не указаны, используется внутренная таблица перекодировки, которая определяет перекодировку из 1025 в 1251.
    Если указан режим -binary оригинальные данные остаются неизменными.
    F:\>unmove.rex JESSY.INSTALL INSTALL
    021I Data set JESSY.INSTALL from SVSKKK
    022I Data set attributes: DSORG=PO,RECFM=FB,LRECL=80,BLKSIZE=800
    018I Member INSTALL extracted to INSTALL

  4. Data sets unloaded by IEHMOVE suitable for network transfer due to fixed record format. However, such files used very rearly. Наборы данных, выгруженные IEHMOVE, пригодны для передачи по сети, так как имеют фиксированную длину записи (80). Однако, на практике этот формат используется очень редко.
