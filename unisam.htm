Function: Функция: 
  Load data set from IEBISAM image. Загрузка набора данных из файла, выгруженного IEBISAM.
 
Syntax:Формат: 
 
  UNISAM [options] iebisam file
 
Options:Режимы: 
 -binary | -text
-ebcdic codepage
-ascii codepage
 
PrerequestedТребования: 
 
  • Either Open object REXX (OOREXX) or Regina REXXOpen Object REXX (OOREXX) либо Regina REXX
  • File extension .rex must be properly associatedРасширение имени файлов .rex должно быть правильно ассоциировано
 
InstallationУстановка 
  Installation procedure not required. Какая-либо специальная установка не требуется.
 
Usage:Применение 
 
  1. Procedure UNISAM has two mandatory parameters - iebisam - name of IEBISAM image and file - name of the restored file. Процедура UNISAM имеет два обязательных параметра iebisam имя файла с образом набора данных, выгруженного IEBISAM и file - имя восстановленного файла.
  2. Data translation. If option -text specified (this is the default option), data would be translated from EBCDIC to ASCII. The options -ascii and -ebcdic may be used to specify a particular code pages for translation. If both -ascii and -ebcdic are missed, an internal translation table is used. This internal table specified 1025 to 1251 translation.
    If option -binary specified, original data remains unchanged.     Перекодировка данных. Если указан режим -text (этот режим активен по уолчанию), option), данные транслируются из кода EBCDIC в код ASCII. Режимы -ascii и -ebcdic могут использоваться чтобы определить, какие именно кодовые таблицы используются при перекодировке. Если режимы -ascii и -ebcdic не указаны, используется внутренная таблица перекодировки, которая определяет перекодировку из 1025 в 1251.
    Если указан режим -binary оригинальные данные остаются неизменными.
